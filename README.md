# Global Retail Analytics

Academic project from BSc Data Science & Business Analytics,
University of London (SIM).

## Overview
Conducted exploratory data analysis and visualisation on a global 
retail dataset using Tableau to surface insights on profitability, 
customer behaviour, and supply chain trends.

## Files
- ST2187_CW.twb — Tableau workbook with interactive dashboards
- Business_analytics_report.docx — written analysis and findings
- Business_analytics_slides.pptx — presentation slides
